# camera_driver
ROS driver for Camera Devices

## Prerequisite
 * Ubuntu 16.04
 * ROS Kinetic
 
## How to Fetch
` $ cd ~/catkin_ws/src`

`$ git clone`

## How to Build
`$ cd ~/catkin_ws`

`$ catkin build camera_driver`

## How to Run
`$ rosrun camera_driver camera_driver_node`

## How to Test
`$ rostopic list`
